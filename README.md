# WriteAdmin
本插件修改自(https://github.com/gogobody/SimpleAdmin)[https://github.com/gogobody/SimpleAdmin] ，目前修改内容较少，黑暗模式仍然有很多需要改进的地方<br>
⚠️注意：本插件的出发点是自用插件，主要是修改黑暗模式的样式。
## 插件特色
### 黑暗模式改进
原作者的是以紫色作为黑暗模式的主颜色，在其基础上我修改为更加契合浏览器及系统的灰黑色(#2c2c2c)<br>
